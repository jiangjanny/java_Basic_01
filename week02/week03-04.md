## 數據庫概念
1. 數據庫 - 存儲數據的倉庫，數據是有組織的進行存儲，簡稱 DataBase (DB)
2. 數據庫管理系統 - 操縱和管理數據庫的大型軟件 ， DataBase Management System(DBMS)
3. SQL - 操作關係數據庫的編成語言，定義一套操作數據庫統一標準， Structred Query Language (SQL)

## SQL 
### SQL 通用語法
1. SQL 語句可以單行或多行書寫，以分號結尾
2. SQL 語句可以使用空格 / 縮進還增加語句的可讀性
3. MySQL 數據庫的SQL 語句不區分大小寫，關鍵字建議使用大寫
   - 單行註解: -- 註解內容 或 # 註解內容 (MySQL 特有)
   - 多行註解: /* 註解內容 */
  
### SQL 分類
1. DDL (Data Definition Language) : 數據定義語言，用來定義數據庫對象(數據庫 / 表 / 字段) <img width="532" height="117" alt="image" src="https://github.com/user-attachments/assets/2bbfd858-6349-447f-a840-e55d2698f5ea" />

2. DML (Data Manipulation Language) : 數據庫操作語言，用來對數據庫表中的數據進行曾刪改
3. DQL (Data Query Language) : 數據查詢語言，用來查詢數據庫中表的紀錄
4. DCL (Data Control Language) : 數據控制語言，用來創建數據庫用戶，控制數據庫的訪問權限

#### DDL - 數據庫操作
1. 查詢 
   - 查詢所有數據庫  : SHOW DATABASES;
   - 查詢當前數據庫 : SELECT DATABASE();
2. 創建
   - CREATE DATABASE [IF NOT EXISTS] 數據庫名 [DEFAULT CHARSET 字符集] [COLLATE 排序規則];
3. 刪除
   - DROP DATABASE [IF EXISTS] 數據庫名;
4. 使用
   - USE 數據庫名;
### DDL - 表操作- 查詢
1. 查詢當前數據庫所有表 : SHOW TABLES;
2.  查詢表結構 : DESC 表明;
3.  查詢指定表的建表語句 : SHOW CREATE TABLE 表名;

### DDL - 表操作-創建
在 SQL 的 **DDL（Data Definition Language）** 中，`CREATE TABLE` 用於建立新資料表。以下是完整語法與範例說明。

---

#### 🔹 基本語法

```sql
CREATE TABLE 表名 (
    欄位1 資料型別 [限制條件],
    欄位2 資料型別 [限制條件],
    ...
);
```

---

#### 🔹 常見資料型別

| 型別           | 說明     |
| ------------ | ------ |
| INT          | 整數     |
| VARCHAR(n)   | 可變長度字串 |
| CHAR(n)      | 固定長度字串 |
| DATE         | 日期     |
| DATETIME     | 日期時間   |
| DECIMAL(m,n) | 精確數值   |
| BOOLEAN      | 布林值    |

---

#### 🔹 常見限制條件（Constraints）

| 限制             | 說明          |
| -------------- | ----------- |
| PRIMARY KEY    | 主鍵          |
| NOT NULL       | 不可為空        |
| UNIQUE         | 唯一值         |
| DEFAULT        | 預設值         |
| AUTO_INCREMENT | 自動遞增（MySQL） |
| FOREIGN KEY    | 外鍵          |

---

#### 🔹 範例 1：建立基本資料表

```sql
CREATE TABLE students (
    id INT PRIMARY KEY,
    name VARCHAR(50) NOT NULL,
    birthday DATE,
    email VARCHAR(100) UNIQUE
);
```

---

#### 🔹 範例 2：含 AUTO_INCREMENT（MySQL）

```sql
CREATE TABLE users (
    user_id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) NOT NULL,
    password VARCHAR(100) NOT NULL,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

---

##### 🔹 範例 3：含外鍵

```sql
CREATE TABLE orders (
    order_id INT AUTO_INCREMENT PRIMARY KEY,
    user_id INT,
    order_date DATE,
    FOREIGN KEY (user_id) REFERENCES users(user_id)
);
```

---

#### 🔹 如果資料表已存在避免錯誤

```sql
CREATE TABLE IF NOT EXISTS table_name (
    ...
);
```
