# 基本數學運算函數

在 SQL 中，**數值函數（Numeric Functions）**用來對數字資料做計算、取整、四捨五入、取餘數、指數運算等。不同資料庫（如 MySQL、PostgreSQL、Microsoft SQL Server、Oracle Database）支援的函數略有差異，但**核心數值函數大致相同**。

## 整理（常見 SQL 數值函數）

| 函數         | 功能     |
| ---------- | ------ |
| ABS()      | 絕對值    |
| MOD()      | 取餘數    |
| POWER()    | 次方     |
| SQRT()     | 開根號    |
| ROUND()    | 四捨五入   |
| CEILING()  | 無條件進位  |
| FLOOR()    | 無條件捨去  |
| TRUNCATE() | 截斷小數   |
| EXP()      | e 的次方  |
| LOG()      | 對數     |
| LOG10()    | 10為底對數 |
| RAND()     | 隨機數    |
| SIGN()     | 判斷正負   |
| PI()       | 圓周率    |
| SIN()      | 正弦     |
| COS()      | 餘弦     |
| TAN()      | 正切     |


---

## 一、基本數學運算函數

## 1️⃣ `ABS()` 絕對值

**用途：** 回傳數字的絕對值。

```sql
SELECT ABS(-10);
```

結果：

```
10
```

實務例子：

```sql
SELECT ABS(balance - 1000) AS difference
FROM accounts;
```

---

## 2️⃣ `MOD()` 取餘數

**用途：** 回傳除法的餘數。

```sql
SELECT MOD(10, 3);
```

結果：

```
1
```

實務例子（判斷奇偶數）：

```sql
SELECT id
FROM numbers
WHERE MOD(id, 2) = 0;
```

---

## 3️⃣ `POWER()` 指數運算

**用途：** 計算次方。

```sql
SELECT POWER(2, 3);
```

結果：

```
8
```

---

## 4️⃣ `SQRT()` 開根號

```sql
SELECT SQRT(16);
```

結果：

```
4
```

---

# 二、取整數與四捨五入

## 5️⃣ `ROUND()` 四捨五入

**用途：** 將數值四捨五入到指定小數位。

```sql
SELECT ROUND(12.345, 2);
```

結果：

```
12.35
```

實務例子：

```sql
SELECT ROUND(price, 0)
FROM products;
```

---

## 6️⃣ `CEILING()` / `CEIL()` 無條件進位

```sql
SELECT CEILING(4.2);
```

結果：

```
5
```

---

## 7️⃣ `FLOOR()` 無條件捨去


結果：

```
4
```

---

## 8️⃣ `TRUNCATE()` 截斷小數（不四捨五入）

```sql

```

以下整理 **SQL 面試 / 證照最常考的數學函數**，每個都用「主題 + 範例 + 說明」，讓你一看就知道怎麼用。
（大部分資料庫如 **MySQL / PostgreSQL / SQL Server / Oracle** 都有這些函數）

---

# 1️⃣ ABS — 取絕對值

**用途**：把負數變成正數。

### 範例

```sql
SELECT ABS(-10);
```

**結果**

```
10
```

### 實務範例

計算 **訂單金額差異**

```sql
SELECT ABS(price - cost) AS profit_diff
FROM products;
```

意思：
計算 `price` 與 `cost` 的差距，不管正負。

---

# 2️⃣ ROUND — 四捨五入

**用途**：將數字四捨五入到指定小數位。

### 範例

```sql
SELECT ROUND(3.14159, 2);
```

**結果**

```
3.14
```

### 實務範例

計算平均分數

```sql
SELECT ROUND(AVG(score), 1) AS avg_score
FROM students;
```

結果：平均分數保留 **1位小數**

---

# 3️⃣ CEILING / CEIL — 無條件進位

**用途**：往上取整數。

### 範例

```sql
SELECT CEILING(3.2);
```

**結果**

```
4
```

### 實務範例

計算 **需要幾頁**

假設一頁10筆資料：

```sql
SELECT CEILING(COUNT(*) / 10.0) AS total_pages
FROM orders;
```

---

# 4️⃣ FLOOR — 無條件捨去

**用途**：往下取整數。

### 範例

```sql
SELECT FLOOR(3.9);
```

**結果**

```
3
```

### 實務範例

計算 **完整箱數**

```sql
SELECT FLOOR(total_items / 12) AS full_boxes
FROM warehouse;
```

意思：
每箱12個，算可以裝滿幾箱。

---

# 5️⃣ MOD — 取餘數

**用途**：計算除法的餘數。

### 範例

```sql
SELECT MOD(10, 3);
```

**結果**

```
1
```

### 常見面試題：判斷奇偶數

```sql
SELECT id
FROM numbers
WHERE MOD(id, 2) = 0;
```

結果：
只找 **偶數**

---

# 6️⃣ POWER — 次方

**用途**：計算次方。

### 範例

```sql
SELECT POWER(2, 3);
```

**結果**

```
8
```

意思：

```
2^3
```

### 實務範例

複利計算

```sql
SELECT principal * POWER(1 + rate, years) AS future_value
FROM investments;
```

---

# 7️⃣ SQRT — 開根號

**用途**：計算平方根。

### 範例

```sql
SELECT SQRT(16);
```

**結果**

```
4
```

### 實務範例

距離計算（常見於 GIS / Data）

```sql
SELECT SQRT(POWER(x2-x1,2) + POWER(y2-y1,2)) AS distance
FROM points;
```

---

# 8️⃣ RAND — 隨機數

**用途**：產生 0~1 的隨機數。

### 範例

```sql
SELECT RAND();
```

**結果**

```
0.348291
```

### 常見面試題：隨機抽資料

```sql
SELECT *
FROM users
ORDER BY RAND()
LIMIT 5;
```

意思：

隨機抽 **5筆資料**

# ⭐ SQL 面試最常考 20 題

---

# 1️⃣ 找最高薪水

```sql
SELECT MAX(salary)
FROM employees;
```

---

# 2️⃣ 找薪水第二高的人

（經典面試題）

```sql
SELECT MAX(salary)
FROM employees
WHERE salary < (
    SELECT MAX(salary)
    FROM employees
);
```

---

# 3️⃣ 找薪水前三高

```sql
SELECT *
FROM employees
ORDER BY salary DESC
LIMIT 3;
```

---

# 4️⃣ 每個部門最高薪水

```sql
SELECT department, MAX(salary)
FROM employees
GROUP BY department;
```

---

# 5️⃣ 每個部門員工數量

```sql
SELECT department, COUNT(*)
FROM employees
GROUP BY department;
```

---

# 6️⃣ 找員工數量 > 5 的部門

（考 **HAVING**）

```sql
SELECT department, COUNT(*)
FROM employees
GROUP BY department
HAVING COUNT(*) > 5;
```

---

# 7️⃣ 找薪水高於平均的人

```sql
SELECT *
FROM employees
WHERE salary > (
    SELECT AVG(salary)
    FROM employees
);
```

---

# 8️⃣ 找重複資料

假設 email 重複

```
users
-----
id
email
```

```sql
SELECT email, COUNT(*)
FROM users
GROUP BY email
HAVING COUNT(*) > 1;
```

---

# 9️⃣ 刪除重複資料（保留一筆）

```sql
DELETE FROM users
WHERE id NOT IN (
    SELECT MIN(id)
    FROM users
    GROUP BY email
);
```

---

# 🔟 找沒有訂單的客戶

（LEFT JOIN 經典題）

```
customers
orders
```

```sql
SELECT c.*
FROM customers c
LEFT JOIN orders o
ON c.id = o.customer_id
WHERE o.id IS NULL;
```

---

# 1️⃣1️⃣ INNER JOIN

```sql
SELECT c.name, o.order_id
FROM customers c
INNER JOIN orders o
ON c.id = o.customer_id;
```

意思：

只找 **有訂單的客戶**

---

# 1️⃣2️⃣ LEFT JOIN

```sql
SELECT c.name, o.order_id
FROM customers c
LEFT JOIN orders o
ON c.id = o.customer_id;
```

意思：

所有客戶 + 訂單

---

# 1️⃣3️⃣ RIGHT JOIN

```sql
SELECT c.name, o.order_id
FROM customers c
RIGHT JOIN orders o
ON c.id = o.customer_id;
```

意思：

所有訂單

```sql
SELECT department, AVG(salary)
```

# 2️⃣0️⃣ 找連續登入的人

（高階題）


```sql
SELECT user_id, COUNT(*)
FROM logins
GROUP BY user_id
HAVING COUNT(*) >= 3;
```
