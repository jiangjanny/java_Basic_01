# MySQL 字符串函數練習題

## 前期準備
1. 準備資料表

    ```java
    CREATE TABLE users (
      id: INT,
      name: VARCHAR(50),
      email: VARCHAR(100),
      city: VARCHAR(50),
      salary: DECIMAL(10,2)
    )
    ```

    插入資料

    ```java
    INSERT INTO users VALUES
    (1,' alice ','alice@gmail.com','taipei',52000.5),
    (2,'BOB','bob@yahoo.com','taichung',43000),
    (3,'Charlie','charlie@hotmail.com','kaohsiung',61000),
    (4,'david','david_lin@gmail.com','tainan',47000);
    ```

## 練習題


###  基礎題

#### 1️⃣ 拼接姓名與城市

使用 `CONCAT()` 顯示：

```
alice from taipei
```

欄位名稱：

```
user_info
```

 ```java
SELECT CONCAT(name, "-", city) AS first_name
FROM users
 ```

---

#### 2️⃣ 使用 `CONCAT_WS()`

輸出：

```
alice-gmail.com
```

提示
把 `name` 和 `email網域` 用 `-` 連接。

---
 ```java
SELECT CONCAT_WS('-', name, SUBSTRING_INDEX(email,'@',-1)) AS user_info
FROM users;
 ```

> 備註 :
```
SUBSTRING_INDEX(str, delimiter, count)
```
| 參數        | 說明    |
| --------- | ----- |
| str       | 原始字符串 |
| delimiter | 分隔符   |
| count     | 取第幾段  |

```sql
SELECT SUBSTRING_INDEX('a,b,c', ',', 1);
```
結果： a  ( 取 第一個 , 前面的內容 )

```sql
SELECT SUBSTRING_INDEX('a,b,c', ',', -1);
```
結果： a  ( 取 最後一段 )

<hr>

#### 3️⃣ 截取 email 前 5 個字

使用：

```
SUBSTRING()
```

輸出欄位：

```
email_part
```
> 語法說明
```
SUBSTRING(str, start, length)
```
| 參數     | 說明                        |
| ------ | ------------------------- |
| str    | 要截取的字符串                   |
| start  | 從第幾個字開始（MySQL 從 **1** 開始） |
| length | 截取幾個字                     |

```sql
SUBSTRING(email,1,5)
```
結果： 
```
email
從第1個字開始
取5個字
```

<hr>
---

#### 4️⃣ 取 name 前 3 個字

使用：

```
LEFT()
```
> 語法說明
```
SELECT LEFT(name, 3) AS name_part
FROM users;
```
| 參數     | 說明      |
| ------ | ------- |
| str    | 字符串或欄位  |
| length | 取左邊幾個字符 |

```sql
SELECT LEFT('Hello',2);
```
結果： 
```
He
```
> 注意：第一筆資料有空格 ' alice '，所以會取到空格。
> 如果想 先去掉空格再取 3 個字，可以這樣寫：

```sql
SELECT LEFT(TRIM(name), 3) AS name_part
FROM users;
```
---

| length | 從右邊取幾個字符 |

```sql

```

lo

### 第二部分：長度與大小寫

#### 6️⃣ 計算 name 長度

輸出：

```
name_length
```

使用：

```
CHAR_LENGTH()
```
> 語法說明 字符長度（對多字節字符，如中文，能正確計算）
```
SELECT CHAR_LENGTH(TRIM(name)) AS name_length
FROM users;
```

---

#### 7️⃣ city 轉成大寫

使用：

```
UPPER()
```

```SQL
SELECT UPPER(city) AS city_upper
FROM users;
```
---

#### 8️⃣ name 轉成小寫

使用：

```
LOWER()
```
```SQL
SELECT LOWER(city) AS city_upper
FROM users;
```

```
TRIM()
```

```SQL

```

#### 🔟 把 email 中的 `gmail` 改成 `google`

使用：

```
REPLACE()
```
```sql
SELECT REPLACE(email, 'gmail', 'google') AS email_upper
FROM users;
```

#### 11️⃣ 找出 name 中字母 `a` 的位置

使用：

```
INSTR()
```
```sql
SELECT INSTR(name, 'a') AS a_position
FROM users;
```
---

#### 12️⃣ 查找 email 中 `@` 的位置

使用：

```
LOCATE()
```

```sql
SELECT LOCATE('@', email) AS at_position
FROM users;
```
> 語法說明

```sql
LOCATE(substr, str)
```
| 參數     | 說明            |
| ------ | ------------- |
| substr | 要查找的子字符串      |
| str    | 原始字符串         |
| start  | 可選，從第幾個位置開始查找 |

返回值：

第一次出現的位置（從 1 開始）

如果找不到，返回 0

### 第四部分：進階

#### 13️⃣ 重複 city 兩次

例如：

```
taipeitaipei
```

使用：

```
REPEAT()
```
```sql
SELECT REPEAT(city, 2) AS city_repeat
FROM users;
```
> 語法說明

```sql
REPEAT(str, count)
```
| 參數    | 說明         |
| ----- | ---------- |
| str   | 要重複的字符串或欄位 |
| count | 重複的次數（整數）  |

返回值：將字符串重複 count 次。
