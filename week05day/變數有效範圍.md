# Fields and Local Variable (Scope)
在Java當中，變數的有效範圍被稱為 Scope。
A Java field 是 a variable inside a class。Fields的scpe 是整個Class。每個 field 可以設定modifer為public或private。

在mothod 內declare 的variable只能由大括號之間的代碼內部進行訪問。這些varables 僅在本地可被辨識，因此稱為局部變量(local variable)。

在 for loop 內部寫的 for(int i = 0 ...) 中的i 也是一個local variable。


你說得對 👍
在 **Java** 裡，**Field 的 Scope 不只是「整個 Class」**，還會受到 **modifier（存取修飾詞）**影響，例如 `public`、`private` 等。下面重新整理一次，比較完整。

---

# 1️⃣ Field（成員變數）

**Field** 是在 **Class 裡、method 外宣告的變數**。

```java
class Student {
    public String name;
    private int age;
}
```

這裡：

* `name`
* `age`

都是 **Field**。

---

# 2️⃣ Field 的 Scope（有效範圍）

Field 的使用範圍會由 **modifier（存取修飾詞）**決定。

常見的 modifier：

| Modifier    | 可以存取的範圍        |
| ----------- | -------------- |
| `public`    | 所有 class 都可以   |
| `private`   | 只有同一個 class    |
| `protected` | 同 package + 子類 |
| *(default)* | 同 package      |

---

# 3️⃣ 範例說明

### public field

```java
class Student {
    public String name;
}
```

其他 class 可以直接使用：

```java
Student s = new Student();
s.name = "Tom";
```

因為 `name` 是 **public**。

---

### private field

```java
class Student {
    private int age;
}
```

其他 class **不能直接使用**：

```java
Student s = new Student();
s.age = 20; // ❌ compile error
```

因為 `age` 是 **private**。

通常要用 **getter / setter**：

```java
class Student {
    private int age;

    public void setAge(int a) {
        age = a;
    }

    public int getAge() {
        return age;
    }
}
```

---

# 4️⃣ Local Variable（區域變數）

Local Variable 是在 **method 裡宣告的變數**。

```java
void calculate() {
    int x = 10;
}
```

特點：

* 只能在 **method 裡使用**
* **不能加 modifier**

下面是錯的：

```java
void test() {
    public int x = 10; // ❌ 不合法
}
```

---

# 5️⃣ Field vs Local Variable

| 特性       | Field                    | Local Variable |
| -------- | ------------------------ | -------------- |
| 宣告位置     | class 裡                  | method 裡       |
| modifier | 可以 (`public`, `private`) | 不可以            |
| scope    | 由 modifier 決定            | method 或 block |
| 預設值      | 有                        | 沒有             |

---

# 6️⃣ 簡單記憶

```
Field → 可以有 modifier (public / private)

Local Variable → 沒有 modifier
```

---

✅ **Java 設計原則**

在 **Object-Oriented Programming** 裡通常會：

```
Field → 設為 private
Method → public
```

例如：

```java
class Student {
    private int age;

    public int getAge() {
        return age;
    }
}
```

這叫 **封裝 (Encapsulation)**。

---

**Variable Shadowing（變數遮蔽）**

很好 👍 我用一個**最常見、最容易考試出現的例子**解釋 **Variable Shadowing（變數遮蔽）**，在 **Java** 裡非常重要。

---

# 1️⃣ 什麼是 Variable Shadowing

**Variable Shadowing** 指的是：

> **Local variable 和 Field 使用同一個名字，Local variable 會「遮住」Field。**

也就是：

```
Local variable > Field
```

Local variable 會優先被使用。

---

# 2️⃣ 最經典範例

```java
class Student {

    int age = 20; // Field

    void show() {
        int age = 10; // Local variable

        System.out.println(age);
    }
}
```

執行：

```
10
```

原因：

```
method 裡的 age → 遮住 class 裡的 age
```

---

# 3️⃣ 如果要用 Field 怎麼辦？

要用 **`this` keyword**。

```java
class Student {

    int age = 20;

    void show() {
        int age = 10;

        System.out.println(age);       // Local variable
        System.out.println(this.age);  // Field
    }
}
```

輸出：

```
10
20
```

---

# 4️⃣ this 是什麼？

`this` 代表 **目前的 object（物件）**。

例如：

```java
this.age
```

意思是：

```
這個 object 的 age
```

---

# 5️⃣ Constructor 裡最常見

這是 **Java 新手最常看到的寫法**。

```java
class Student {

    int age;

    Student(int age) {
        this.age = age;
    }
}
```

解釋：

```
this.age  → Field
age       → Parameter (Local variable)
```

意思是：

```
把 parameter age 存到 field age
```

---

# 6️⃣ 圖解（超好記）

```
class Student
{
    int age = 20;   ← Field

    void show()
    {
        int age = 10;   ← Local variable

        age        → 10
        this.age   → 20
    }
}
```

---

# ⭐ Java 初學者記住這三句

```
Field 在 class
Local variable 在 method
Local variable 會 shadow Field
```

如果要用 **Field**：

```
this.fieldName
```
