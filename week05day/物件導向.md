# 1️⃣ Constructor 基本概念

當你建立一個 class 的物件時：

```java
User user = new User();
```

`new User()` 這個動作就會 **呼叫 Constructor**。

---

# 2️⃣ 最基本的 Constructor

```java
class User {

    String name;

    // Constructor
    public User() {
        name = "Guest";
    }

}
```

使用：

```java
public class Main {
    public static void main(String[] args) {

        User user = new User();

        System.out.println(user.name);
    }
}
```

輸出：

```
Guest
```

流程：

```
new User()
   ↓
呼叫 Constructor
   ↓
name = "Guest"
```

---

# 3️⃣ Constructor 的規則

### 1️⃣ 名字必須和 class 一樣

```java
class User {

    public User() { }  // ✔ 正確

}
```

錯誤：

```java
public void User() { }  // ❌ 這就變普通 method
```

---

### 2️⃣ 沒有 return type

Constructor **不能寫回傳型別**

❌ 錯

```java
public void User() {}
```

✔ 正確

```java
public User() {}
```

---

### 3️⃣ 建立物件時自動執行

```java
User u = new User();
```

等於

```
呼叫 User Constructor
```

---

# 4️⃣ Constructor 帶參數 (最常用)

可以在建立物件時 **直接帶資料**

```java
class User {

    String name;
    int age;

    public User(String name, int age) {
        this.name = name;
        this.age = age;
    }

}
```

使用：

```java
User user = new User("Tom", 20);

System.out.println(user.name);
System.out.println(user.age);
```

輸出：

```
Tom
20
```

---

# 5️⃣ `this` 的用途

```java
public User(String name, int age) {
    this.name = name;
    this.age = age;
}
```

意思：

```
this.name = 物件的 name
name = 傳進來的參數
```

如果沒有 `this`

```java
name = name;
```

就會變成

```
參數 = 參數
```

物件不會被賦值。

---

# 6️⃣ Default Constructor（預設建構子）

如果你 **沒有寫 constructor**

Java 會自動幫你建立一個：

```java
public User() {}
```

但只要你寫了一個 constructor：

```java
public User(String name) {}
```

Java **就不會幫你產生預設 constructor**。

所以這樣會錯：

```java
User user = new User(); // ❌ error
```

必須寫：

```java
public User() {}
```

---

# 7️⃣ 多個 Constructor（Overloading）

可以有多個 constructor。

```java
class User {

    String name;
    int age;

    public User() {
        name = "Guest";
        age = 0;
    }

    public User(String name) {
        this.name = name;
    }

    public User(String name, int age) {
        this.name = name;
        this.age = age;
    }
}
```

使用：

```java
User a = new User();
User b = new User("Tom");
User c = new User("Tom", 20);
```

當資料庫查詢後：


就可以直接建立物件。

# 1️⃣ User Model（資料模型）

通常我們會先建立 **User.java**

```
src
 └─ org.example
     └─ model
         └─ User.java
```

```java
package org.example.model;

public class User {

    private int id;
    private String name;
    private String email;

    // Constructor
    public User(int id, String name, String email) {
        this.id = id;
        this.name = name;
        this.email = email;
    }

    // Getter
    public int getId() {
        return id;
    }

    public String getName() {
        return name;
    }

    public String getEmail() {
        return email;
    }
}
```

這裡的 constructor：

```java
public User(int id, String name, String email)
```

作用是：

```
建立 User 物件
並且把資料初始化
```

# 2️⃣ DAO 從 DB 取資料

假設 DB 有一筆資料

```
id | name | email
1  | Tom  | tom@gmail.com
```

DAO 會查詢 DB

```
src
 └─ dao
     └─ UserDao.java
```

```java
package org.example.dao;

import org.example.model.User;

import java.sql.Connection;
import java.sql.ResultSet;
import java.sql.Statement;

public class UserDao {

    public User getUser() throws Exception {

        Connection conn = DBConnection.getConnection();

        Statement stmt = conn.createStatement();

        ResultSet rs = stmt.executeQuery("SELECT * FROM users LIMIT 1");

        if (rs.next()) {

            int id = rs.getInt("id");
            String name = rs.getString("name");
            String email = rs.getString("email");

            // 使用 constructor 建立物件
            User user = new User(id, name, email);

            return user;
        }

        return null;
    }
}
```

重點在這行

```java
User user = new User(id, name, email);
```

這就是 **constructor 最常見用途**

```
DB資料
   ↓
ResultSet
   ↓
Constructor
   ↓
Java Object
```

# 6️⃣ Constructor vs Setter

Constructor 初始化

```java
User user = new User(1,"Tom","tom@gmail.com");
```

Setter 初始化

```java
User user = new User();
user.setId(1);
user.setName("Tom");
user.setEmail("tom@gmail.com");
```

工程師通常：

```
Constructor → 建立資料
Setter → 修改資料
```


```java
}
```
