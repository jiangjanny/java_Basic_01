# 靜態屬性以及方法（`static`）

在 **Java** 中，`static` 關鍵字主要用於 **記憶體管理**。

`static` 可以讓 **同一個 Class 的所有物件共享同一個變數或方法**。

另外，我們也可以 **直接透過 Class 本身來存取 static 變數或呼叫 static 方法**，而不需要先建立物件。

這個關鍵字被稱為 **Static（靜態）**，因為它不是動態的。
這意味著它對於該 **Class 的所有 Object 都是相同的一份資料**。

因此：

* 所有 `static variable` 和 `static method`
* **都屬於 Class 本身，而不是某一個 Object**

當我們使用 `static variable` 或 `static method` 時：

* **不需要先建立物件（instantiate object）**

例如 Java 內建的 **`Math` 類別**就有很多 `static` 方法：

```java
double result = Math.sqrt(16);
System.out.println(result); // 4.0
```

這裡我們沒有建立 `Math` 物件，
而是直接使用 **Class 名稱呼叫 static method**。

---

# Static Variable（靜態變數）

靜態變數會被 **所有物件共享**。

```java
class Student {
    static String school = "NTU";
}
```

所有 `Student` 物件都會共用同一個 `school` 變數。

---

# Static Method（靜態方法）

靜態方法 **屬於 Class，而不是物件**。

```java
class Calculator {

    static int add(int a, int b) {
        return a + b;
    }

}
```

呼叫方式：

```java
int result = Calculator.add(3, 5);
```

# `private`（存取修飾子）

`private` 是一種 **Access Modifier（存取修飾子）**。

如果一個變數或方法被宣告為 `private`：

* **只能在同一個 Class 裡面使用**
* 其他 Class **無法直接存取**

例如：

```java
class Person {

    private int age;

}
```

這樣 `age` 只能在 `Person` 類別內使用。

通常我們會搭配 **getter / setter** 來存取：

```java
class Person {

    private int age;

    public int getAge() {
        return age;
    }

    public void setAge(int age) {
        this.age = age;
    }

}
```

✅ **簡單記憶方式**

| 關鍵字       | 功能               |
| --------- | ---------------- |
| `static`  | 屬於 Class，共享給所有物件 |
| `private` | 只能在同一個 Class 中使用 |
